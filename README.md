# Sales Data Exploration

## Dataset
Kaggle- Sample Superstore Sales Dataset (Retail)

## Objective
Explore sales data, clean it, calculate statistics, and visualize trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab


## Analysis Performed
- Data inspection and basic cleaning
- Duplicate and missing value checks
- Total sales and average order calculation
- Sales by category analysis
- Sales trend over time visualization

## Key Insights
- Technology category generates the highest sales
- Sales fluctuate over time with seasonal trends
- Average order value provides pricing insights

## Project Files
- `sales_data_exploration.ipynb` -> Analysis Notebook
- `Sample - Superstore.csv` -> Dataset
- `Sales By Category.png` -> Visualization Output 1
- `Sales Trend Over Time.png` -> Visualization Output 2
